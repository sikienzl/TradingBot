Trading Bot Pi Release Notes - v1.0.20

Date: 2026-05-06

Highlights
- Updated the AI co-pilot cadence to a month-safe 62-minute interval.
- Prevented the default 720-call monthly cap from expiring early in 31-day months.
- Aligned repository defaults and deployment templates with the live Raspberry Pi configuration.
- Deployed the updated bot configuration and code to the Raspberry Pi host.

Operational Impact
- The Mammouth AI co-pilot remains active, but now spaces calls slightly wider to fit the monthly cap.
- A configured limit of 720 monthly calls now lasts across 31-day months.
- New deployments inherit the safer interval automatically from code defaults and example env files.

- .env.example
- .env.live.example

Verification Checklist
- Trading bot starts cleanly after deployment.
- AI co-pilot interval is set to 62 minutes in the live environment.

Notes
